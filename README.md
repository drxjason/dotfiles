# My dotfiles

written guide to my dotfiles for people who don't wanna parse my files

## ZSH

```bash
\_OwO_/
jason@host ~ %
```

### Plugins

- zsh-autosuggestions (package install)
- zsh-syntax-highlighting (package install)

### Aliases

- `ls='ls --color=auto'`

## Vim

### Editor Settings

- mouse mode on
- line numbers
- syntax highlighting
- 4 spaces as tabs
- auto indentation
- highlighted search

### Vim plugins (vim-plug)

- fzf (fuzzy finder)
- vim-airline (fancy tabline)
- nerdtree (tree explorer)
